# Author headshots

Drop square author headshots here as `<slug>.jpg` (or .png/.webp).

Naming: lowercase, hyphenated, derived from author full name.
- "Edward Sam"  -> `edward-sam.jpg`
- "Robert Smith" -> `robert-smith.jpg`

Recommended:
- Square crop, 400x400 minimum
- Real headshot or AI-generated avatar

The `<AuthorByline>` component falls back to a brand-green monogram
circle when no photo is supplied. Pass `photoSrc="/authors/edward-sam.jpg"`
explicitly per post once images are uploaded.
